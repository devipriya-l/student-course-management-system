# Learning Management System (LMS) - Django

A web-based Learning Management System (LMS) developed using **Django** that helps manage students, courses, and academic information efficiently through a simple and interactive dashboard.

This system allows administrators to perform operations such as adding students, viewing records, managing courses, and monitoring data through a clean user interface.

---

## Features

- User authentication (Login / Logout)
- Dashboard with statistics cards
- Add and manage student details
- Course management
- Navigation using Bootstrap navbar
- Secure session handling
- Database integration using Django ORM
- Responsive UI using Bootstrap

---

## Tech Stack

**Backend**
- Python
- Django

**Frontend**
- HTML
- CSS
- Bootstrap

**Database**
- SQLite (default Django database)
